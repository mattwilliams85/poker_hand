POKER HAND
==============================================
Readme:
Run the program for an array of five poker cards. Returns the type of hand.

H = hearts         A = ace
C = clubs          K = king
S = spades         Q = queen
D = diamonds       J = jack

(e.g. poker_hand([5S,4H,KC,2C,2C]) >> will return >> "two of a kind")
